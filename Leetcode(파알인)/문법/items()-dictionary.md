
# ✍️ items()

딕셔너리 {key: value ...} -> ```tuple: [(key, value) ...]```
```python
car = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

x = car.items()

>>> print(x)
dict_items([('brand', 'Ford'), ('model', 'Mustang'), ('year', 1964)])
```
